清理已核销的旧记录

    

      仅删除 已使用 的行，减小库表体积；未使用的码不会删。填 0 表示清空全部已核销记录（不按天数）；填正整数则只删「使用时间」早于「今天 − N 天」的记录。订单里仍保存兑换码明文，删表行不影响订单记录。建议先预览再确认删除。
    

    

      若预览结果里出现 Not Found / 404：说明当前 YUNOTE_API_BASE 指向的主服务版本过旧，还没有清理接口；需部署最新代码，或本地启动主服务并把 API 地址指到本机。
